## 1. 游戏工具函数测试 (game-utils)

### TC-GU-001: random() 函数
| 项目 | 内容 |
|------|------|
| **测试目标** | 验证随机数生成在指定范围内 |
| **前置条件** | 设置固定随机种子 |
| **测试步骤** | 1. 调用 random(1, 10) 20次<br>2. 检查返回值范围 |
| **预期结果** | 所有返回值在 [1, 10] 范围内，且为整数 |
| **优先级** | P0 |

### TC-GU-002: chance() 概率函数
| 项目 | 内容 |
|------|------|
| **测试目标** | 验证概率触发逻辑 |
| **测试步骤** | 1. 测试 chance(1) 应总是成功<br>2. 测试 chance(0) 应总是失败<br>3. 测试 chance(0.5) 1000次，统计成功率 |
| **预期结果** | 1. 100% 成功<br>2. 0% 成功<br>3. 成功率在 40%-60% 之间 |
| **优先级** | P0 |

### TC-GU-003: calculateDamage() 伤害计算
| 项目 | 内容 |
|------|------|
| **测试目标** | 验证伤害计算逻辑 |
| **测试步骤** | 1. 攻击20，防御5，应造成约15点伤害<br>2. 防御大于攻击时，至少造成1点伤害<br>3. 暴击时伤害增加50% |
| **预期结果** | 1. 伤害 ≈ 15 × (0.9~1.1)<br>2. 伤害 ≥ 1<br>3. 暴击伤害 = 基础伤害 × 1.5 |
| **优先级** | P0 |

### TC-GU-004: checkDodge() 闪避检查

|------|------|
| **测试目标** | 验证闪避率计算 |

## 测试优先级说明

| 优先级 | 说明 | 测试类型 |
|--------|------|----------|
| P0 | 核心功能，必须100%通过 | 单元测试、集成测试 |
| P1 | 重要功能，应高通过率 | 单元测试、集成测试 |
| P2 | 辅助功能，允许部分失败 | 单元测试 |
